strchr()

  • strchr() function is useful to find the first occurrence of a given character in a string. It starts counting from 0 onwards. For example

        char *n = strchr(string, ch);

    This function returns a character type pointer that contains memory address of the character 'ch' in the string. If the character is not found in the string, then this function returns NULL which is the last character in the memory.

  • Let us take a string 'str' which contains 'puzzle'. There are 6 characters in this string, each character takes 1 Bye memory. If the string starts at a memory address 1000, the first character 'P' is stored at this address 1000. The next character 'u' is stored at 1001, 'z' is stored at 1002, and so on. See in the figure.

    Fig: strchr() retreiving the position of the character 'z' in 'Puzzle'.

  • Suppose we are searching the position of charaacter 'z' in the string 'Puzzle'. In this example, str is 'Puzzle' and ch is 'z'. Then strchr() function returns the memory address of the character 'z' into the pointer n. According to above figure, it is 1002 So, the character type pointer n represents 1002.

    •     The position of the charcter = n - base address.
                                 = 1002 - str
                                 = 1002 - 1000
                                 = 2

        #include<stdio.h>
    #include<string.h>
