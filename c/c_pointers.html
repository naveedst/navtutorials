Pointers

Pointer:

  • Pointer is a variable which stores the memory address of another variable.

  • To declare a pointers, we should write a star (*) symbol before the variable and declare it as:

        int *pv;

    Thus when * symbol is used before the variable, the computer will be able to understand that it is not an ordinary variable, but a pointer variable. pv is a pointer variable which is declared as integer type.So, pv can be used to store address of an integer type variable. Similarly,

       
    float *x;

    Here, x is a pointer variable which is declared float type. So, it can be used to store the address of another float variable.

  • 'value at' or 'indirection' operator (*) using this operator we can retrieve the value stored at the variable referenced by the pointer. This operator should be written before the pointer as *pv and then it represents the value 15 which is the value at v.

    Program to understand how to create and use a pointer

        #include<stdio.h>
    void main()
    {
       int v=15;
       int *pv;
       pv = &v;
       printf("\n value of v = %d", v);
       printf("\n value of pv =%d", pv);
       printf("\n value of *pv =%d", *pv);
    }

    Output:

        value of v = 15
    value of pv = 65784
    value of *pv = 15

    From the above output, we can understand that v value and *pv values are same. The value of pv which is displayed by this program is 65784. This may differ from computer to computer since it represents the internal memory address.

    • Pointer arithmetic:

  • Let us take two pointers p1 and p2.

  • If pointers represent the addresses of array elements, then it is possible to perform subtraction on the pointers.

        p2 - p1   /* valid */

    If p1 presents the address of the first element and p2 represents address of the last element of the array, then p2 - p1 gives the number of elements in the array.

  • Some cases, subtraction of pointers is not allowed. In the same way we cannot perform addition, multiplication and division operations on pointers.

        p1 * p2          /* invalid */
    p1 + p2 / 3     /* invalid */

  • It is possible to increment or decrement a pointer.

        p1++    /* valid */
    --p2    /* valid */

    A pointer represents the address of the memory block. when it is incremented, it shows the address of the next memory block. Similarly wehn a pointer is decremented, it shows the address of previous memory block.

  • It is possible to add some constant value to a pointer. Similarly it is possible to subtract a constant value from the pointer.

         p1+5           /* valid */
     p2 = p2 - 10   /* valid */

  • We can compare two pointers to know which one is bigger or smaller.

         if(p1 == p2)    /* valid */
     if(p2 >= p1)   /* valid */

  • We can change the value of the variable through 'value at' operator.

        a = 15;
    p1 = &a;
   *p1 = 25;

    Now 'a' value will not be 15, it will 25

    Program to swap two numbers without using pointers.

        #include<stdio.h>
    void swap(int x, int y);
    void main()
    {
       int a,b;
       a=10;
       b=5;
       printf("\n Before swapping -> a = %d, b = %d", a, b);
       swap(a,b);
       printf("\n after swapping -> a = %d, b = %d", a, b);
    }

   void swap(int x, int y)
   {
      int t;
      t = x;
      x = y;
      y = t;
   }

    Output:

        Before swapping -> a = 10, b = 5
    After swapping -> a = 5, b = 10

    In the prorgram swap() function is called and a and b values are passed from the main() function, as:

        swap(a, b);

    In this way, by calling a function by passing the values of variables is called 'call by reference' or 'pass by value'.
