Dynamic memory allocation

Dynamic memory allocation

Dynamic Memory:

  • Dynamic memory is the memory allotted by the programmer at runtime. The size of the dynamic memory grows at runtime and its very huge memory.

    • malloc()

  • This function is useful to allocate required number of bytes on heap memory. This function is defined in the header files <alloc.h> and <stdlib.h>. In unix we should use only <stdlib.h> since <alloc.h> is not available. Using this function, let us allocate 2 bytes of memory, useful to store an integer.

       int *p;
   p = (int *)malloc(2);

    The dynamic memory allocated on heap is referenced by the integer type pointer p. malloc() function returns void type pointer as (void *). This pointer is converted into int type pointer as (int *). This conversion is called 'type casting'. The above statement can also be written as: 

       p=(int *)malloc(sizeof(int));

    sizeof() operator returns the siz of int datatype, i.e. 2 bytes. This size may vary from one operating system to another. To receive data into the memory referenced by p, we can write:

        scanf("%d", p);

    • realloc()

  • realloc() function is useful to modify and reallocate the memory which is already allocated by malloc() function. realloc() function is defined in <alloc.h> and <stdlib.h>. Suppose, we are allocating dynamic memory of 20 byes for a string 'str' using malloc() as:

       char *str;
   str = (char *)realloc(str,25);

    str is a pointer that refers to the allocated 20 bytes. Now, we can modify this memory and reallocate 25 bytes of memory to str as:

       str = (chr *)realloc(str, 25);

    • calloc()

  • calloc() function allows a block of bytes. It is defined in <alloc.h> and <stdlib.h> header files. It allocates a number of blocks of each having a group of bytes. For example to allocate 10 blocks of 2 bytes each, we can write:

        int *ptr;
    ptr= (int *)calloc(10, sizeof(int));

    The size of each integer is 2 bytes which is given by sizeof(int). It will allocate memory to store 10 such integer numbers.

    • free()

  • This function is useful to free the memory which is already allocated using malloc() or realloc() or calloc() functions. free() function is defined in <alloc.h> and <stdlib.h>. To allocate 20 bytes of memory to store the name of a person, we can use malloc() function as:

       char *name;
   name = (char*)malloc(20);

    To free or delete the memory which is allocated through the pointer 'name', we can write:

       free(name);

    • Program to understand how to create static and dynamic memory allocations

        #include<stdio.h>
    #include<alloc.h>
    void main()
    {
       int num;
       int *ptr;

       printf("\n Static memory allocation: ");
       printf("\n Enter a number: ");
       scanf("%d", &num);
       printf("you entered: %d", num);
       printf("\n Dynamic memory allocation: ");
       printf("\n Enter a number:");
       ptr=(int *)malloc(sizeof(int));
       scanf("%d",ptr);
       printf("you entered:%d",*ptr);
       free(ptr);
    }

    Output:

           Static memory allocation:
       Enter a number: 100
       You entered: 100
       Dynamic memory allocation:
       Enter a number: 200
       You entered: 200
