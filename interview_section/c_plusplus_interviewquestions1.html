C++ Interview Questions

C++ Questions and Answers

What is operator overloading?

It gives the capability to the operator to work on different types of operands.

List the access specifiers in C++.

There are three access specifiers in C++ namely Public, Private and Protected.

What are Public, Private and Protected?

Public data members or member functions can be accessed from anywhere, within the class or from outside the class. Private data members cannot be accessed from outside the class. They can only be accessed within the class. The protected members of the base class are accessible only to the derived classes.

How do we create a this pointer?

The ‘this’ pointer gets created when a member function (non-static) of a class is called.

What is the outcome when we call the constructor explicitly?

Whenever the constructor is called explicitly a nameless object gets created.

How do we create an object dynamically?

The keyword ‘new’ allows us to create an object dynamically.

Can constructors return a value? How does it handle error values?

An error value can never be returned from a constructor. An exception alone can be thrown from within the constructor. Constructor is called whenever an object gets created. There can be no situation where we want to return a value at the time of creation of an object.

How do we overload constructors and destructors?

Destructors cannot be overloaded. Constructors can be overloaded, as it is like other member functions of the class. But it will not return any value.

What is the purpose of operator overloading?

It makes C++ operators work with objects and gives new meanings to existing C++ operators.

What is multiple inheritance?

A class that can be derived from more than one base class is called a multiple inheritance.

What is the significance of using Inheritance?

Inheritance permits the reusability of software; derived classes can extend the capabilities of base classes with no need to modify or even access the source code of the base class.

What is an Abstract class?

It is often defined as one that will not be used to create any objects, but exists only to act as a base class of other classes.

What is a friend function?

A friend can access a class private data, even though it is not a member function of the class.

What is a static function?

A static function is one that operates on the class in general rather than on objects of the class. It can operate on static variables.

Why is assignment operator overloaded?

This is necessary when it must do more than merely coping one object’s contents into another

What is a pure virtual function?

It is a virtual function that has no body and is used in a base class.

Write a statement that a member function can use to return the entire object of which it is a member, without creating any temporary objects.

return *this;

Does the expression delete P delete the pointer or the object being pointed to by P?

The expression delete P deletes the object being pointed to by P.

Is it necessary to accept a reference in the copy constructor?

It is essential, because if in the copy constructor an object is accepted by value then the copy constructor will fall in a recursive loop.

Compare static binding and dynamic binding.

When a function call gets resolved at compile-time it is called static binding or early binding. When the call gets resolved at runtime it is called dynamic binding or late binding.

What is a stream?

It is used to represent the flow of data. For different kinds of data, different streams are used.

Name the three parts of an iostream system.

A buffer, a specification system and a translation system.

Compare manipulator and setf( ) function.

The I/O specification class (ios) is at the root of the iostream class hierarchy. Manipulators are the instructions to the output stream to modify the output in various ways. The setf( ) function is used to set the flags of the ios but manipulators directly insert the formatting instructions into the stream.

Compare iterator and iteration.

An iterator is an object that moves through the container accessing each element in the container. The process of moving from element to element in the container is called ‘iteration’.

What is friend class?

It is a class where all the member functions and data members of the class become ‘friends’.

How do we get the information about the object at runtime?

Using typeid( ) operator and dynamic_cast operator.

What is class library?

A group of related classes, supplied as a separate product, is often called a class library.

What are templates?

By using templates, we can design a single class/function that operates on many data types, instead of having to create a separate class/function for each type. When used with functions they are known as ‘function templates’, whereas when used with classes they are called ‘class templates’.

What is the size of an object of an empty class?

One byte.
